# Pre Training 2026: Python Exercises

This repository contains the Python exercises for the 2026 training program.  
Each day’s exercises are in a separate folder.  

---

## Day 1 — Python Basics
- Folder: `Day 1`
- Exercises on **variables, operators, control flow, loops, functions, input, and formatted output**.
- Includes:
  1. `exercise_1.py` — Data types & operators (name, age, coffee habit, salary, retirement, coffee budget)  
  2. `exercise_2.py` — Grade classifier function with user input and fixed scores  
  3. `exercise_3.py` — Multiplication table generator with input validation & right-aligned tables  
- Example outputs are attached in the folder.  
- [Go to Day 1 Exercises](./day-1/README)

---

## Day 2 — Python Functions & Data Processing
- Folder: `Day 2`
- Exercises on **functions, lists, dictionaries, loops, sorting, and formatted tables**.
- Includes:
  1. `exercise_1.py` — Word frequency counter for a hardcoded paragraph  
  2. `exercise_2.py` — Student report generator: averages, grades, topper highlighting, sorted output  
- Example outputs are attached in the folder.  
- [Go to Day 2 Exercises](./day-2/README)

---

## Day 3 — Python OOP & File I/O
- Folder: `Day 3`
- Exercises on **object-oriented programming, file input/output, JSON persistence, CLI, and error handling**.
- Includes:
  1. `tasks.py` — Command-line task tracker:
     - Add, complete, delete, and list tasks
     - Tasks are persisted in `tasks.json`
     - Handles missing arguments, invalid IDs, and corrupt JSON files
- Example usage is documented in the folder README.  
- [Go to Day 3 Exercises](./day-3/README)

---
